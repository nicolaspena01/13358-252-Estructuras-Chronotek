# Binary Search Tree - Recorridos y Eliminación

## Autor
**Nicolás Arenas**

## Descripción
Implementación de un Árbol Binario de Búsqueda (BST) en Java con los recorridos:
- In-Order
- Pre-Order
- Post-Order  
y el método **Delete** basado en el pseudocódigo visto en clase.

## Ejemplo de datos
Data = {8, 3, 10, 1, 6, 14, 4, 7, 13}
