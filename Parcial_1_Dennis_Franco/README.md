## Descripción breve
Solución en Java para gestionar información de estudiantes y demostrar:
- Ordenamientos: Insertion Sort (por `age` asc), Selection Sort (por `semester` desc) y Bubble Sort (por `socioEconomicStratum` asc).
- Búsquedas: Linear Search (por `programId`) y Binary Search (por `age`, con arreglo previamente ordenado por edad).
